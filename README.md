# 🤖 Intelligent Build & Release Manager

An AI-powered web application that generates production-ready HTML applications from user prompts. The project leverages the GROQ API and the Llama 3.1 model to automate code generation through a modern and responsive interface.

---

## 📖 Project Overview

Developing applications from scratch can be time-consuming, especially for repetitive UI development. This project simplifies the process by allowing users to describe their requirements, after which the AI generates clean and structured HTML code instantly.

The application is built with React and Vite, providing a fast and interactive user experience while integrating the GROQ API for intelligent code generation.

## ⚡ How It Works

1. 🔑 Configure your GROQ API key.
2. 💬 Enter the application requirements as a prompt.
3. 🤖 The prompt is sent to the GROQ API.
4. ⚙️ The AI processes the request and generates HTML code.
5. 📄 The generated code is displayed instantly for review and use.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/intelligent-build-and-release-manager.git
```

2. Open the project in Visual Studio Code.

3. Install dependencies

```bash
npm install
```

4. Create a `.env.local` file and add your GROQ API key

```env
VITE_GROQ_API_KEY=your_api_key_here
```

5. Start the development server

```bash
npm run dev
```

6. Open the application in your browser.
